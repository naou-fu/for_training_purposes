# 🪨 📄 ✂️ Rock-Paper-Scissors Web Project

Welcome to my first web-based game! This is a simple, interactive Rock-Paper-Scissors game built with JavaScript.

## 🎮 How to Play

Follow these steps to get the game running on your computer:

### 1. Prerequisites
You will need **VS Code** (Visual Studio Code) installed on your machine. If you don't have it, [download it here](https://visualstudio.com).

### 2. Setup the Environment
To see the game in action, you need the **Live Server** extension:
* Open VS Code.
* Go to the **Extensions** view (click the square icon on the left or press `Ctrl+Shift+X`).
* Search for **"Live Server"** and click **Install**.

### 3. Launch the Game
1. Open your project folder in VS Code.
2. Locate the `index.html` file in the file explorer.
3. **Right-click** on `index.html`.
4. Select **"Open with Live Server"**.
